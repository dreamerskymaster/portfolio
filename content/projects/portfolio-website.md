# ManuFX Portfolio Website

## Overview
This website itself is a testament to my ability to bridge the gap between manufacturing engineering and modern software development. It serves as a dynamic platform to showcase my work, skills, and professional journey.

## Technical Highlights
*   **Modern Stack**: Built using React 18, TypeScript, and Vite for blazing fast performance.
*   **Dynamic Content**: Implemented a custom Markdown parsing engine to allow for easy content updates without code changes.
*   **Interactive UI**: Utilized Framer Motion for smooth page transitions and engaging micro-interactions.
*   **Responsive Design**: Fully optimized for all devices, from mobile phones to large desktop screens.
*   **Performance**: Optimized for speed and SEO, achieving top scores in web vitals.

## Design Philosophy
The design reflects my professional identity: clean, precise, and innovative. The "ManuFX" brand combines "Manufacturing" with "Effects/Efficiency," symbolizing the transformative impact of technology on industrial processes.
