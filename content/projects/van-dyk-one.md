# Van Dyk One Mobile App

## Overview
Van Dyk One is an ambitious initiative to bring all field operation tools into a single, unified mobile experience. Currently in development, this app aims to empower technicians with instant access to resources, regardless of connectivity.

## Key Features
*   **Offline Mode**: Critical for technicians working in remote facilities with poor internet.
*   **Unified Toolset**: Access to DykScribe, RAG System, and Spare Parts ordering in one app.
*   **Cross-Platform**: Built with React Native for seamless deployment on iOS and Android.

## Status
*   **Current Phase**: Development
*   **Projected Launch**: Q1 2026
